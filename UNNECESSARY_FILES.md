# Unnecessary Files in Repo

These files/folders should **not** be tracked in version control or can be excluded.

## Should be ignored (add to .gitignore)

| Path | Reason |
|------|--------|
| `venv/` | Virtual environment – reproducible via `pip install -r requirements.txt` |
| `__pycache__/` | Python bytecode cache – auto-generated |
| `scanner/__pycache__/` | Same as above |
| `.DS_Store` | macOS folder metadata |

## Optional – generated/runtime

| Path | Reason |
|------|--------|
| `scan_results.log` | Runtime log – grows over time, can be regenerated |
| `non_fno_stocks.txt` | Generated by API – can be recreated via `/api/v1/symbols/non-fno/generate` |

## Keep in repo

- `main.py` – Flask entrypoint
- `generate_non_fno_stocks.py` – NSE symbol generator
- `requirements.txt` – dependencies
- `README.md` – documentation
- `postman_collection.json` – API examples
- `strategy_logic.txt` – strategy reference
- `scanner/` – core package (all .py files)
